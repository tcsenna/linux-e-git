# Aprofundando no Git II

A turma aprendeu a criar a conta no Github e fazer sincronizações com repositório remoto.

Comandos derivados de:

- git rm

- git checkout

- git merge

- git remote add

- git reset

- git push


e muito mais- foram discutidos.
